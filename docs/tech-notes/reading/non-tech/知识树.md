## 人生篇

### 知行合一，心决定行

- 不幸都是自己的选择，不符合预计的行为根本都出在心上
- 目的导向：立足于现在的目的而不是过于的原因 = 活在当下，不忆过去，不忧未来
- 没有勇气导致了不幸：不是你不能改变。人无论在何时也无论处于何种环境中都可以改变。你之所以无法改变，是因为自己下了"不改变"的决心。要想改变生活方式需要很大的"勇气"。面对变化产生的"不安"与不变带来的"不满"，你一定是选择了后者。
- 知行合一：不是你先知道怎么上王者，然后才打上王者。而是你知道怎么上王者的时候你就已经是王者了。知道做不到等于不知道
- 放下执念：当你想赢而不在乎赢的时候，就会迎接胜利。享受每一次抉择，注重自己专注当下，不要太执着于结果。也就是固化的诚心，夫随其成心而师之
- 第一性原理：拨开表象看本质，也就是说达到最终目的，并不是只有一条路

### 目的论

思考问题的时候应该多采取目的论：不应该立足过去的原因，而是现在的目的。

例如：愤怒是达成目的的一种手段，你之所以愤怒是因为你想要用更加快捷的方式使并不抵抗的对方屈服。

心理创伤并不存在。决定我们自身的不是过去的经历，而是我们自己赋予经历的意义。

### 人生时期

**低谷期：**

- 不要相信任何人画得饼
- 没有利益冲突的时候不要反驳任何人
- 封闭自己，与自己无关的事情不问，不想，不多嘴
- 停止无意义的社交
- 每天留点独处和反思的时间

**不要沉溺于安逸：** 安逸会毁掉一个人，从命理的角度，除了特殊的格局，不需要官、杀出现，大多数人都是需要有一点官杀的，有了官杀，才会对自己有要求，对自己有要求的人，才会自律，才会成长，人在任何时候都要保持不断地成长，否则你的能力、智商各个方面都会退化，人是在"动"中变化成长，一直处于"静"的状态，只会让人停滞不前。佛系是强者看淡名利后的随意，是弱者不思进取享受安逸的借口。

### 坚守的道理

**断舍离：** "不舍得"和"念旧"是改变命运的最大牵绊之一，勇于舍弃和改变

**拒绝纠结：** 纠结和算计的心态反而得不到好的结果，弱势思维的人都有一个通病，就是喜欢墨迹，纠结，无论是做事上，还是选择上，人际关系上，而越强的人，反而越简单，越干脆，越果断，从命理的角度性格确实决定了命运，因为不同性格的人，在面对同一件事情，同一个选择的时候，会因为性格导致完全不同的结果。

## 人际篇

### 课题分离：自己 | 他人

**接收：** 警惕他人的赞美和捧杀，不要Care别人对你的评价而是在乎自己内心的想法，大众认为的正确不会让你变得更好，不要为了满足别人的期待而生活，要有钝感力，不要敏感

**输出：** 对底层的人不要过于客气，学会拒绝，不要做免费垃圾桶

**方法论：** 沟通机制：删减、扭曲、类推

**课题分离：** 一切人际关系矛盾都起因于对别人的课题妄加干涉，或者自己的课题被别人妄加干涉。

**对人应该：** 不刻意展现自己的好，要放下我正在付出的自我感动。

## 幸福之道

**祛魅：** 要想过得好，要学会祛魅，核心思想"你很优秀，跟我有什么关系"。分为两层，对人和对世界。对人祛魅就是对于比自己优秀的人不要放大优点，不要过滤缺点，不要盲目尊重。对世界祛魅就是不要悲观，要保持乐观，享受世界的真实，拒绝理想主义，接受世界的真实。

**钝感力：** 也就是不要敏感，共情力不要太强。共情力太强很容易感知别人的痛苦/快乐。比如身边的人不顺心找你吐槽，就不需要设身处地为别人着想，表面上你温柔善良心思细腻，但是时间长了就会一直为别人撑伞，忽视自己才是淋雨的人。越同情谁就会背负他人的背负的东西。适度的同情是悲悯，过度的同情是灾难。

**不要取悦别人：** 人一旦习惯了取悦他人，就有一直取悦不完的人。但是如果学会了取悦自己，那么自然会不断有新人来取悦自己。人生是一条很长的路，起起伏伏，人生不会因为一个单一的时间点被毁掉，人生的容错率无限大。因此心要大一点，得罪一些人，做错一些事没什么大不了。不要满足别人的期待去生活

**被人讨厌不是坏事：** 比起别人如何看自己，我更关心自己过得如何。也就是想要自由地生活。

**不要晒幸福：** 不要晒成功，不要晒幸福，不要晒恩爱。自己的成功自己看到就好，越少人知道越好。

**只筛选不改变：** 不管是友情还是爱情都不要改变，只要筛选。最傻的就是想改变别人的人。不要觉得烂人烂事值得培养和慢慢磨合，到最后一边忍受痛苦一边还要将痛苦合理化。要知道一个人根深蒂固的思想和毛病是改变不了的，要么接受，要么换人。长时间的积压，总会有爆发的那一天。

**物极必反：** 越自负的人越自卑。如果有人骄傲自大，那一定是因为他有自卑感。

**拒绝内耗：** 不甘心是一种情绪内耗，人在情绪激动的时候容易极端，容易忽略掉别人的好，加重自己的单方面情绪，任何一段感情都能让人成长，取其精华去其糟粕，忘掉不美好的，只留下好的，你会更容易释然，拥有过，经历过，都是一种体验，时间会冲淡一切。

**减少倾诉：** 别人知道你的秘密越少，你就越安全。

### 阅读

**阅读的三要素：** 目的，价值，意义

阅读的目的一定是改变，没有改变的阅读没有意义，只有基于改变的阅读才会带来提升。输出倒逼输入。

**阅读目标的三个层次：**

1. **第一层：** 建立底层知识逻辑体系（也就是我目前的目标层次）
2. **第二层：** 在你的领域里，通过阅读把它做到最好（工作中的指导层次）
3. **第三层：** 价值变现，也可以是给我们人带来的价值观的提升（离我还远得很）

**阅读的反馈设计**

阅读反馈设计的3个要素：过去、现在、未来

- **过去（反思读过的内容）：** 回忆和串联
- **现在（评估当前状态）：** 掌握程度
- **未来（规划后续行动）：** 目标和实践

### 习惯

**培养好习惯的方法：**

- **明确目标：** 具体化目标，制定可以衡量的目标
- **有利环境：** 包括环境设计和减少诱惑，总之就是让动作尽可能的轻易去达成
- **持续行动：** 将习惯融入生活之中，比如做完某一件熟悉的事后去做想要的事
- **反馈机制：** 正向反馈和负向反馈，可以记录进展。可以让想做的事情联动我们喜欢/不喜欢做的事情
- **耐心与毅力：** 半途而废的根本原因是自己的认知没跟上，在你心中的分量不够

### 结构化思维

思维的锻炼就是锻炼自己看待一个问题时，会从哪些角度，分成哪些方面去看。可以来自于经验，也可以来自于套路或者说技巧。

**切名词：** 就是划分，比如简历上的个人优势可以拆分成专业性，广度性，深度性，前瞻性。

**四大原则：**

- 数字说话
- 洞见优于表象

**对思维的反思**

问自己三个问题：

1. 第一个问题：我这么思考的依据是什么？
2. 第二个问题：这么思考会产生什么样的结果？
3. 第三个问题：产生的结果是否是我想要的？

## 理性消费习惯准则（7条行动框架）

### 1. 质量价值优先法则

**核心逻辑：** 单次使用成本 = 商品价格 ÷ 有效使用次数

**执行标准：**

- ✔️ 服装类：选择缝线密度≥12针/3cm的工艺，天然纤维含量>70%的面料
- ✔️ 电子产品：优先3年质保期以上机型，参考ifixit可维修评分≥7分的设备
- ✔️ 工具类：通过ASTM/EN国际标准认证产品（如扳手需符合DIN 3110标准）

### 2. 需求验证决策树

**购买前必问：**

- ▢ 是否解决现存痛点？（如冬季缺保暖外套）
- ▢ 替代方案成本是否更高？（租用/借用/改造旧物）
- ▢ 闲置风险等级：高频（每周3次+）/中频/低频（季度1次-）

**防营销陷阱：** 识别"伪需求创造"（如第5支同色系口红）

### 3. 生命周期成本核算

**决策启示：** 高频使用品侧重长期节能性，低频品侧重基础质量

### 4. 消费心理防御机制

**锚定效应破解：**

- ✘ 原价¥599现价¥199 → 计算真实折扣率（66% off）
- ✔️ 横向比价：查询历史价格曲线（用「慢慢买」等比价工具）

**多巴胺延迟策略：** 将商品放入购物车，72小时后再决策

### 5. 可持续消费分层

**消费金字塔：**

- **基础层（必要支出）：** 选择B Corp认证品牌
